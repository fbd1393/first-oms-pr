1- Start a django project with command (python django-admin startproject projectname).
2- After that check the settings file.
3- Create apps with command (python manage.py startapp appname).
4- Add the app to the settings file.
5- Make a url file for the app.
6- Include the app's url file to the main url file.
    for doing this we need to import Include from django.urls and import (from . import views)
7- Add the views tha we want in views file. Then make a path for any views that we have.
8- Import render from django.shortcuts or import HttpResponse from django.http to the views file.
9- Make and add template to the views.
10- Django template's path structure in apps is...
    --appname
    ----templates
    -------appname
    ----------template-name1.html
    ----------template-name2.html
    ----------template-name3.html

11- Static folder is a file that we store all of our images and styles and js,...
    Static folder is in root directory 
    after we create static folder we need to add it in settings.py file
    
    STATICFILES_DIRS = [
    os.path.join(BASE_DIR, 'static')
    ]
    static
    --styles
    --js
    --images
    --,...

12- For using scss in whit python we have 2 ways 
    a- we can use extensions like sass live server for vscode.
    b- use pip install libsass for adding the libsass package.
